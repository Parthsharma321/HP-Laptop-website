# HP 245 G10 – Product Landing Page

A premium, interactive product landing page for the **HP 245 14-inch G10** business notebook — built to showcase specifications, performance benchmarks, connectivity, and a live configurator in a modern, cinematic web experience.

🔗 **Live Demo:** [hp-245-g10-notebook-pc.vercel.app](https://hp-245-g10-notebook-pc.vercel.app/)

---

## 📌 Overview

This project is a fully responsive, single-page marketing website for the HP 245 G10 laptop. It combines rich product storytelling with functional UI elements like a benchmark comparison chart, an interactive port hotspot diagram, a photo gallery with full-screen preview, customer testimonials, an FAQ accordion, and a live price configurator.

> *Demo Website Concept — created for design/development portfolio purposes and not an official HP product.*

---

## ✨ Features

- **Hero Section** – Bold introduction with call-to-action buttons (`Explore Features`, `View Gallery`)
- **Overview Section** – Highlights of the AMD Ryzen™ processor lineup, display, battery, security, and connectivity
- **Benchmark Comparison** – Visual performance comparison across Office Productivity, Content Creation, and Multi-Tasking workloads (Ryzen™ 7 7730U vs Ryzen™ 5 7530U vs previous-gen Intel Core i5)
- **Technical Specifications Grid** – Detailed breakdown of:
  - Processor & iGPU Graphics
  - Memory (RAM) & Storage
  - Display Panel
  - Battery Backup
  - Weight & Chassis
  - Webcam & Audio
  - Wireless Network
  - Security (HP Wolf Security, TPM 2.0)
  - Ports Suite
  - Operating System options
- **Interactive Connectivity Hub** – Hover/tap port hotspots on a laptop diagram to reveal bandwidth and capability details
- **Gallery** – Lifestyle and product imagery with full-screen preview modal
- **User Reviews** – Testimonial cards with reviewer photos, names, and roles
- **FAQ Accordion** – Common customer questions (RAM upgrade, multi-monitor support, MS Office bundling, warranty, gaming capability, security, fast charging)
- **Live Configurator** – Real-time price calculator for:
  - Processor (Ryzen™ 5 / Ryzen™ 7)
  - RAM (8GB / 16GB / 32GB)
  - Storage (256GB / 512GB / 1TB NVMe SSD)
  - Operating System (Windows 11 Home / Pro)
  - Auto-updating configuration summary and estimated total price (₹)
- **Sticky Navigation** – Smooth-scroll links to all major sections
- **Fully Responsive Design** – Optimized for desktop, tablet, and mobile

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v16 or higher recommended)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/Parthsharma321/HP-Laptop-website.git

# Navigate into the project directory
cd HP-Laptop-website

# Install dependencies
npm install

# Run the development server
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000) (or the appropriate port) in your browser.

### Build for Production

```bash
npm run build
```
